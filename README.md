# Daily Data Structures & Algorithms Practice (Java)

This repository is dedicated to weekly practice of Data Structures and Algorithms using Java.
Each week will focus on a specific data structure followed by relevant algorithms.

## Structure
The project is organized by weeks in `src/main/java/weekX`.
Each package contains a `Main.java` to demonstrate the implementation.

## Weekly Plan

### Week 1: Stack
- **Topic**: Stack Data Structure
- **Implementation**: Array-based Stack (`MyStack.java`)
- **Status**: Completed

### Week 2: Queue (Planned)
- implementation of Queue

### Week 3: Linked List (Planned)
- singly/doubly linked list

## How to Run
You can run the main method for each week from your IDE or terminal.
Example for Week 1:
```bash
javac src/main/java/week1/stack/*.java
java -cp src/main/java week1.stack.Main
```
